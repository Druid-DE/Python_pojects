# Salary counter 🧮

## 📊 Калькулятор зарплаты на Python

Данная программа считает зарплату сотрудников, которая складывается из оклада, премии и бонуса, а так же выводит статистику: зарплату каждого сотрудника, общий зарплатный фонд и среднюю зарплату.
Премия зависит от занимаемой должности, а бонус выбирается каждому сотруднику индивидуально.
