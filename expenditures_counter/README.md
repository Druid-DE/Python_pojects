# Expenditures counter 🧮

## 📊 Калькулятор расходов на Python

Калькулятор расходов, который:
- Предлагает ввести данные по расходам
- Считает расходы по каждой категории
- Сравнивает все ваши расходы с лимитом по расходам
- Выводит статистику расходов и дает оценку 

### 🚀 Быстрый старт

```bash
# 1. Скопируй этот код в Pycarm или на сайт 
https://online-python-compiler.ru

# 2. Введи значения расходов через запятую
Например 1000, 2000, 3000, и т.д.

# 3. Укажи свой желаемый лимит по расходам
Например 20000

# 4. Программа выведет статистику ваших расходов
Сумму обязательных расходов, расходов на продукты,
расходов на развлечения и общую сумму расходов, а
так же покажет уложились вы в свой лимит или нет.
```
### 📈 Данная программа поможет вам оценить уровень своих расходов и понять какие траты вам стоит урезать чтобы уложиться в бюджет!
